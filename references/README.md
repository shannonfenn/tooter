# References

This folder stores local copies of study resources. Keep source URLs and licensing notes close to the downloaded files.

- `aata/`: official Judson AATA PDFs and generated source-diff reports.
- `courses/pitt-math0430/`: David Roe's Pitt Math 0430 pages, problem sets, and solutions.
- `courses/mit-18.703/`: MIT OCW 18.703 lecture notes, assignments, and exams.

Refresh with:

```sh
uv run python scripts/fetch_references.py
```
