# Resource Map

## AATA

- Official AATA page: https://twjudson.github.io/aata.html
- Current HTML edition: linked from the official page, updated August 1, 2025.
- Runestone version: linked from the official page, updated weekly and includes interactive Sage cells.
- Official source: https://github.com/twjudson/aata
- 2021 PDF: `references/aata/pdfs/aata-2021.pdf`
- 2025 PDF: `references/aata/pdfs/aata-2025.pdf`

The official page says the next print edition is planned for August 2027, then every four years.

## Pitt Math 0430

Primary exercise subset.

- Fall 2017 course page: https://math.mit.edu/~roed/courses/430/
- Spring 2016 course page: https://math.mit.edu/~roed/courses/430_S16/

Prefer Fall 2017 first. Use Spring 2016 when a chapter or topic is not covered by Fall 2017 or when an extra worked solution is useful.

## MIT OCW 18.703

Secondary reference.

- Course home: https://www.ocw.mit.edu/courses/18-703-modern-algebra-spring-2013/
- Readings page maps lectures to Judson sections.
- Lecture notes are useful for alternate exposition.
- Assignments are mixed Herstein/Judson and should be treated as optional backup.
